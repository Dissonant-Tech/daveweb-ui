About Me

It all started with the HTC HD2. It was a Windows Mobile 6.5 phone released in 2010. The specs at the time were pretty great, but the OS was clunky and killed any performance the hardware had to offer. So I decided to try something out: Flashing an Android ROM.

I really enjoyed the new OS, and the performance was amazing compared to the previous speeds, but most of all, I was hooked. From there I started to tinker with Android ROMS and the AOSP in general. In that vein, I also started to read up on Java programming, so that I could tinker further with my new found hobby.

2010-2012

Once I had learned I could extend an OS with my own tools (something) that had never occurred to me previously while using operating systems such as Windows) I started to yearn for an OS more feature rich and versatile, so I began to branch out and see what other operating systems I could run on my phone. Thats when I learned you could run an entire Linux OS on your Android phone, and connect to the local VNC client using any VNC app on your phone. It was clunky, and a bit awkward, but nevertheless, I was hooked once more.

I then started to install Linux on my desktops as well. Ubuntu 12.04 was the first if I remember correctly. I found Linux amazing, the guts of the operating system were right there for you to explore, all organized as files. I became a surgeon, dissecting and analyzing different parts of the system, as well as patching new programs and scripts. I learned and fell in love with the CLI and now use the command line extensively in my day-to-day.

2012-2014

Of course, with my dive into Linux and customization I began also to dive deeper into programming and scripting. I started using python more and more, as well as listening to programming and Linux related podcasts. Where I first got the Idea to setup my own website and blog as a personal project.

My first website was a personal challenge of sorts. I wanted to deploy something from scratch, no wordpress, no heroku, no firebase, etc. I spun up a VPS (a DigitalOcean droplet) and got to work setting up django with uwsgi (and later gunicorn), nginx, postgres, and using `git push` to deploy my code.

2014-Present
